# 🚀 FairLens AI

AI-powered fairness auditing platform for detecting and mitigating bias in decision systems.

---

## 🌐 Live Demo
👉 https://fairlens-ai.vercel.app

---

## 🧠 Problem

AI systems used in hiring, lending, healthcare, and public services often introduce hidden bias.

---

## 💡 Solution

FairLens AI provides a complete workflow:

- Detect bias
- Explain decisions (Gemini AI)
- Suggest mitigation
- Enable safe deployment

---

## ⚡ Features

- 🔍 Bias Scan  
- 🤖 Gemini Explanations  
- 🔁 Decision Replay  
- 🛠 Mitigation Planner  
- 📊 Audit Reports  
- 📡 Live Monitoring
